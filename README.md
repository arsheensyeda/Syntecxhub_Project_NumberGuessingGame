This is a Number Guessing Game built using Python.
The program randomly selects a number, and the user must guess it with the help of higher/lower hints.

The project demonstrates Python concepts such as loops, conditionals, functions, and the use of the random module.
----
FEATURES


- Random number generation

- Difficulty levels:

Easy (1–50)

Medium (1–100)

Hard (1–200)

- Higher / Lower hints

- Attempt counter

- Replay option

- Best score tracking (lowest attempts in session)

- Structured using functions for clean logic separation
